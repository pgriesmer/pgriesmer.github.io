Just Shoe Me Search Page

CS1300 Project, Fall 2018

Overview

As part of my UI/UX course at Brown this fall, I was tasked with using React to develop a webpage that could filter and sort a dozen different items. The page I built could eventually be used as a results page for an online men's shoe store.

1 // Components

I began by making three components: one for the site header, one for each product that would appear on the results page, and one for the list that would contain these products.

Header component

Product component

List component

2 // Filters

I thought an online shoe store could distinguish itself by indicating when users might wear different shoes. To this end, I added an "Occasion" field to each product and gave users the ability to filter by the same.

Occasion filter

I then added a second filter so that users could filter out only certain materials, search functionality that serves as a dynamic filter, and a "Reset Filters" button to reset both the occasion and material filters to their default state.

Material filter

Search filter

3 // Sorting

I also gave users the ability to sort by ascending or descending price. By default, I configured it so that footwear is sorted by ascending price.

Sort by descending price

Outcomes

Final product

The filters, search, and sorting outlined above work in conjunction with one another on the final page.

Users can sort by price descending and then filter down to "Canvas" shoes only and would only be shown canvas footwear in price-descending order.

As someone whose primary JavaScript background revolves around AngularJS, this project helped me adjust to the paradigm needed to work with React.

If I had more time, I would add the ability to sort by occasion (i.e., dressiness) as well. I would also change the favicon to something more representative of the site and tweak the amount of whitespace I'm using a bit more until it looks just right.
